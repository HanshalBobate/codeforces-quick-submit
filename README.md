# Codeforces Quick Submit

Adds a native Quick Submit button directly to Codeforces problem pages.

## Features

- One-click submission
- Automatically detects problem code
- Auto-fills submission form
- Lightweight
- Native Codeforces styling

## Example

Problem URL:

https://codeforces.com/problemset/problem/977/A

Automatically fills:

977A

on the submission page.
